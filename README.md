## Installation

Using curl
```bash
bash <(curl -s https://raw.githubusercontent.com/Caxvalencia/dotfiles/main/install)
```

Using wget
```bash
bash <(wget -qO- https://raw.githubusercontent.com/Caxvalencia/dotfiles/main/install)
```
